
# Excalidraw Clone – Web Drawing Application

This project is a **web-based Excalidraw clone** that allows users to create, edit, and manage freehand diagrams and shapes on an interactive canvas.  
It focuses on **real-time drawing, clean UI, and essential whiteboard features**.

## Features

- Secure **Login & Signup** flow
- Interactive **drawing canvas**
- Draw shapes: rectangles, circles, arrows, freehand
- Select, move, resize, and delete shapes
- Toolbar with drawing tools and controls
- Save canvas data to backend
- Export drawings locally
- Clean and responsive UI

---

## Core Functionality

### Authentication
- User login & registration
- Session-based access to canvas

### Canvas & Drawing
- Shape creation and manipulation
- Mouse-based interactions
- Layered drawing support

### Actions
- Save canvas state
- Export drawings
- Reset canvas
